# .NET Core React Starter

## Prerequisites

* [Git](https://git-scm.com/)
* [NodeJs](https://nodejs.org/en/)
* [.NET Core](https://www.microsoft.com/net/core)

## Get started

```shell
sh install.sh
sh start.sh
```

Browse to http://localhost:5000/
